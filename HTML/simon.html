Game description

Simon is an electronic game of memory skill invented by Ralph H. Baer and Howard J. Morrison with software programming by Lenny Cope. The device creates a series of tones and lights and requires a user to repeat the sequence. If the user succeeds, the series becomes progressively longer and more complex. Once the user fails or the time limit runs out ( only the former in our case) the game is over. It gets its name from the simple children's game of Simon Says, but the gameplay is based on Atari's unpopular Touch me arcade game from 1974.
